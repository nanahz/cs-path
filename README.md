---

## 📌 项目简介
这是我的**计算机科学（CS）系统学习记录仓库**。  
目标不是「收藏资料」，而是用**公开作品驱动学习**：
- 按领域系统学习 CS 核心知识
- 用笔记、代码、项目输出代替单纯的「看完了」
- 记录学习过程中的踩坑、心得与实践
- 打造一份可追溯、可复用、可对外展示的学习成长档案

---

## 🎯 我的学习目标
1.  构建完整的 CS 知识体系（基础 + 核心 + 应用）
2.  从「只会用工具」升级为「懂原理、能实现」
3.  每个阶段都有可交付的小作品/笔记/代码
4.  长期主义：用 1-2 年时间，把零散知识拼成体系
